# 📝 Notes App

A simple **Notes Application** built using **HTML, CSS, and JavaScript** that allows users to create, edit, save, and delete notes directly in the browser. Notes are stored using **LocalStorage**, so they remain available even after refreshing the page.

---

## 🚀 Features

* Create new notes
* Edit notes directly inside the note box
* Save notes to browser **LocalStorage**
* Delete notes easily
* Notes persist after page refresh
* Simple and clean user interface

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Application logic
* **LocalStorage API** – Persistent storage in the browser

## ⚙️ How It Works

1. Click **Create Notes** to generate a new editable note.
2. Type directly inside the note box.
3. Click **Save** to store notes in LocalStorage.
4. Click the **delete icon** to remove a note.
5. Refresh the page and your saved notes will still be there.
